== Description ==
ContentCalendar is a drag and drop editorial calendar that helps you plan and schedule content in a visual manner.
You can park your ideas for each category on your blog. When ready to write just drag and drop them to the calendar.
#### Features
* Drag and drop Editorial Calendar
* Reminders on upcoming blog posts
* Improves content publishing workflow
* Simplifies content ideation process

#### Coming soon - Premium features:
* Ability to add team members
* Social media publishing
* Ordering content
* Multiple calendars
* Multiple content types
* Team chat
* Task templates

== Installation ==

You can install ContentCalendar via the WordPress.org plugin directory or manually by uploading the files to your server.

= Using WordPress Gallery =

1. Sign in to your WordPress blog
2. Go to 'Plugins' -> 'Add New'
3. Search for 'contentcalendar'
4. Find 'ContentCalendar' in the results and click 'Install Now'
5. Follow the on screen intructions

= Manual Installation =

1. Backup your WordPress database.
2. Download the plugin zip file 'contentcalendar.zip' to your computer.
3. Upload and install through the 'plugins' panel in your WordPress dashboard. You can manually upload 'contentcalendar.zip' to the 'wp-content/plugins/' directory if you prefer.
4. Activate the plugin through the 'plugins' menu in your WordPress dashboard.
5. Sign in with your ContentCalendar account after clicking the ContentCalendar link on the WordPress admin sidebar
